🌍 Analyse de l'Alimentation Mondiale

Contexte
Ce projet présente une analyse de données sur les enjeux mondiaux de l’alimentation, avec un focus sur la sous-nutrition.

Objectifs
- Étudier la disponibilité alimentaire mondiale
- Identifier les pays les plus touchés par la sous-nutrition
- Mettre en évidence les disparités régionales
- Produire des recommandations basées sur les données

Technologies utilisées
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

Données
- Population
- Disponibilité alimentaire
- Aide alimentaire
- Sous-nutrition

Résultats
L'analyse met en évidence les écarts importants entre la disponibilité alimentaire et les besoins nutritionnels dans plusieurs régions du monde.
