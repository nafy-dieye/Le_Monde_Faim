🌍 Analyse de l'Alimentation Mondiale

## Contexte
Ce projet présente une analyse de données sur les enjeux mondiaux de l’alimentation, avec un focus sur la sous-nutrition.

Objectifs
1. Étudier la disponibilité alimentaire mondiale
2. Identifier les pays les plus touchés par la sous-nutrition
3. Mettre en évidence les disparités régionales
4. Produire des recommandations basées sur les données
## Méthodologie

1. Collecte des données
2. Nettoyage et préparation
3. Analyse exploratoire (EDA)
4. Visualisation
5. Interprétation des résultats

## Technologies utilisées
1. Python
2. Pandas
3. NumPy
4. Seaborn
5. Jupyter Notebook

## Données
1. Population
2. Disponibilité alimentaire
3. Aide alimentaire
4. Sous-nutrition

## Résultats
L'analyse met en évidence les écarts importants entre la disponibilité alimentaire et les besoins nutritionnels dans plusieurs régions du monde.
